Selium is a blazing fast programmable data platform, written in Rust. Use Selium to build realtime applications, transform data streams and create resilient microservices.

Why Selium?

  • Selium is composable - Our functional API library allows you to declaratively compose data streams right from your code. Need 2 hours of message retention capped at 100mb with at least once delivery guarantees? You need but ask.
  • Selium has no configuration - Really. None. Express all your requirements at runtime using our composable API.
  • Selium loves WebAssembly - Write WebAssembly modules and colocate them on Selium's router network for dramatically faster TTFB.
  • Selium doesn't do pipelines - Instead of boxing your data in, Selium provides free access to any data, any time. Each client composes their own curated streams, leveraging WebAssembly modules.
